📺 Java Streaming Service Simulation

A backend-focused simulation of a streaming platform built using Java 21, designed to reinforce core programming concepts and modern object-oriented practices.

This project models how a streaming service might handle content, users, and interactions—without relying on external frameworks or databases.

🚀 Features
🎬 Content management (movies, series, etc.)
⭐ Rating logic
📚 Object-Oriented Design (encapsulation, inheritance, polymorphism)
🧠 Use of modern Java features (Java 21)
🗂️ Clean project structure and separation of concerns

🧠 What I Learned
Writing clean and maintainable Java code from scratch
Applying OOP principles to a real-world scenario
Structuring a project without relying on frameworks
Using modern Java features effectively
🧪 Example Use Cases
Add new content to the platform
Simulate a user watching content
Rate or categorize content
List available media
